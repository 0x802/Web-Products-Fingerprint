# Web-Products-Fingerprint
web products fingerprint as json 


### Example:
```json
 {
    "path": "/",
    "request_method": "get",
    "request_headers": {},
    "request_data": "",
    "status_code": 0,
    "headers": {
      "Server":"xxxxxxxx-xxxxx"
    },
    "keyword": [
      "top.location=\"/remote/login\";"
    ],
    "priority": 0,
    "name": "Fortinet VPN"
}
```
